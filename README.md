# Markdown Link Copier

A simple Chrome extension that allows you to copy a selected link in Markdown format: `[Link Text](URL)`.

## Features
- Adds a "Copy as Markdown" item to the context menu when right-clicking links.
- Smartly retrieves link text, even from complex elements.
- Copies directly to clipboard.

## Installation

1. Clone this repository.
2. Open Chrome and navigate to `chrome://extensions`.
3. Enable **Developer mode**.
4. Click **Load unpacked** and select the extension directory.

## Usage

Right-click on any link and select **Copy as Markdown**.
